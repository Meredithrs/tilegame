## Introduction The Snapdragon engine consists of a number of modules. These modules are members of a global object named Snapdragon. ##Snapdragon The global module contains functions for manipulating the game engine. `Snapdragon.initialize()` Initializes the game engine and starts the game. `Snapdragon.addTerrain(tileset)` Injects custom terrain tiles into the game engine. `Snapdragon.addObjects(tileset)` Injects custom object tiles into the game engine. ##Map The Map module contains functions for manipulating the map. `Snapdragon.map.at(x, y)` Returns the items at a particular coordinate in the map. `Snapdragon.map.tileset.addTerrain(terrain_tile)` Adds a terrain tile to the terrain tileset. `Snapdragon.map.tileset.addObject(object_tile)` Adds an object tile to the object tileset. `Snapdragon.map.tileset.getTerrain()` Returns an array containing the terrain tileset. `Snapdragon.map.tileset.getObjects()` Returns an array containing the object tileset. `Snapdragon.map.load(x, y)` Loads the scene at the specified coordinates into the map. ##Player The Player module contains functions for manipulating the player. `Snapdragon.player.getX()` Returns the x position of the player. `Snapdragon.player.getY()` Returns the y position of the player. `Snapdragon.player.teleport(x, y, map)` Instantly moves the player to the specified location if it is a legal move. `Snapdragon.player.isWithin(x, y, distance)` Checks if the player is within a certain distance of the specified coordinates. `Snapdragon.player.move(x, y, map, callback)` Moves the player to the specified coordinates and then invokes a callback. ##Interface The Interface module contains functions for manipulating the interface. `Snapdragon.interface.chat.send(message, color)` Sends a message through the chat box. `Snapdragon.interface.chat.getMessages()` Gets all messages. `Snapdragon.interface.spellbook.addSpell(spell)` Adds a spell to the spellbook. `Snapdragon.interface.spellbook.fill(spells)` Adds an array of spells to the spellbook.
