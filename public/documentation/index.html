Introduction

The TileScape engine consists of a number of modules. These modules are members of a global object named TileScape.

TileScape

The global module contains functions for manipulating the game engine.

TileScape.initialize()

Initializes the game engine and starts the game.

TileScape.addTerrain(tileset)

Injects custom terrain tiles into the game engine.

TileScape.addObjects(tileset)

Injects custom object tiles into the game engine.

Map

The Map module contains functions for manipulating the map.

TileScape.map.at(x, y)

Returns the items at a particular coordinate in the map.

TileScape.map.tileset.addTerrain(terrain_tile)

Adds a terrain tile to the terrain tileset.

TileScape.map.tileset.addObject(object_tile)

Adds an object tile to the object tileset.

TileScape.map.tileset.getTerrain()

Returns an array containing the terrain tileset.

TileScape.map.tileset.getObjects()

Returns an array containing the object tileset.

TileScape.map.load(x, y)

Loads the scene at the specified coordinates into the map.

Player

The Player module contains functions for manipulating the player.

TileScape.player.getX()

Returns the x position of the player.

TileScape.player.getY()

Returns the y position of the player.

TileScape.player.teleport(x, y, map)

Instantly moves the player to the specified location if it is a legal move.

TileScape.player.isWithin(x, y, distance)

Checks if the player is within a certain distance of the specified coordinates.

TileScape.player.move(x, y, map, callback)

Moves the player to the specified coordinates and then invokes a callback.

Interface

The Interface module contains functions for manipulating the interface.

TileScape.interface.chat.send(message, color)

Sends a message through the chat box.

TileScape.interface.chat.getMessages()

Gets all messages.

TileScape.interface.spellbook.addSpell(spell)

Adds a spell to the spellbook.

TileScape.interface.spellbook.fill(spells)

Adds an array of spells to the spellbook.
